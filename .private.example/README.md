# Private Workspace Example

Create a local `.private/` directory with this shape for real restaurant data:

```text
.private/
  branding.yml
  receipts/
  order_guides/
    current/
  seed_data/
  notes/
  agents/
```

Do not commit `.private/` to this repository. If a specific restaurant needs versioned private data, initialize `.private/` as its own private Git repository.

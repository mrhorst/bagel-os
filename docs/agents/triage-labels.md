# Triage Labels

Use the default triage label vocabulary.

| Role | Label |
| --- | --- |
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Ready for an AFK agent | `ready-for-agent` |
| Ready for a human | `ready-for-human` |
| Will not be actioned | `wontfix` |

## Agent Rules

- Do not invent alternate labels for these five roles.
- If labels do not exist in GitHub yet, create or apply these exact strings.
- Keep feature/domain labels separate from triage labels.
